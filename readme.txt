Integrate Smartlook user recording and analytics into your WordPress site with ease. Track user behavior, create heatmaps, and improve conversion rates.

== Description ==

Smartlook is a powerful user recording and analytics platform that helps you understand how visitors interact with your website. This WordPress plugin makes it easy to integrate Smartlook into your site without any coding knowledge.

**Key Features:**

* **Easy Setup** - Simple configuration through WordPress admin
* **User Recording** - Record and replay user sessions
* **Heatmaps** - Visual representation of user interactions
* **Conversion Tracking** - Monitor user behavior and optimize conversions
* **Privacy Compliant** - GDPR and privacy-friendly implementation
* **Multi-language Support** - Built-in translation support
* **Performance Optimized** - Lightweight and fast loading

**What You Can Do:**

* Track how users navigate through your site
* Identify where users get stuck or confused
* Optimize your conversion funnels
* Improve user experience based on real data
* A/B test different page layouts
* Understand mobile vs desktop user behavior

**Perfect For:**

* E-commerce websites
* Business websites
* Landing pages
* SaaS applications
* Marketing agencies
* UX designers and developers

= Installation =

1. Upload the `smartlook` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings > Smartlook' to configure your API key
4. Enter your Smartlook API key from your dashboard
5. Choose your preferred data center region
6. Save settings and start recording!

= Frequently Asked Questions =

= How do I get a Smartlook API key? =

Sign up at [smartlook.com](https://smartlook.com) and get your API key from the dashboard. The plugin will guide you through the setup process.

= Is Smartlook GDPR compliant? =

Yes, Smartlook provides GDPR-compliant user recording with built-in privacy controls and consent management.

= Does this affect my site's performance? =

No, the plugin is lightweight and optimized. Smartlook's recording script loads asynchronously and won't slow down your site.

= Can I exclude certain pages from recording? =

Yes, you can configure exclusions through the Smartlook dashboard or by modifying the plugin code.

= What regions are supported? =

The plugin supports multiple regions: Europe (EU), United States (US), Australia (AU), Canada (CA), Singapore (SG), and Japan (JP).
